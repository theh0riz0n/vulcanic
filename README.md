## installation and startup

1. clone repo
2. establish dependencies:
   ```
   npm install
   ```
3. run app:
   ```
   npm run dev
   ```

## structure

- `/components` - ui components
- `/pages` - next.js pages
- `/lib` - utilities and hooks
- `/styles` - styles and css
- `/public` - static files
